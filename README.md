# 🚨 RapidAid – Emergency Help Locator

> Instantly find nearby hospitals, police stations, and pharmacies when every second matters.

## 📌 Description
RapidAid is a real-time emergency help locator that uses geolocation and interactive maps to help users quickly find nearby hospitals, police stations, and pharmacies during critical situations.

## 🚀 Features
- 📍 Live location detection  
- 🏥 Hospital locator  
- 🚓 Police locator  
- 💊 Pharmacy locator  
- 🗺️ Interactive map
